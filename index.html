6.快速替换

6.1 第一种方法：先选中单词，Ctrl+H,下方出现小窗口

6.2.第二种方法: 用多行游标智能替换。

先选中单词，Ctrl+D,有多少个 “你选中的单词”，就按几下Ctrl+D,就可看到每个单词都有闪烁的光标，这时就可看到 动态替换你选的单词了。 Ctrl+k 跳过你不想选修改的单词，因为 “你选中的单词”在有的地方是不需要替换的。

6.3多行游标 this is a book this is a book this is a book 第一种方法： Ctrl+D 选择中“book” 取消其中一个"book" 按Ctrl+k后接着按Ctrl+D 第二种方法: 选择中“book”,按 ALT+F3 ，这时所有“book”都会有闪烁的光标 第三种方法: Ctrl+A,选中所有，然后Ctrl+Shift+L,这时所有“book”都会有闪烁的光标 第四种方法: 按住Shift,按住鼠标右键,光标闪烁点向下拖拽，可以自由选取每一列，让其所有列都有闪烁光标,这时所有“book”都会有闪烁的光标
